### Add

```sh
yarn add @onesy/ui-react
```

### Use

```javascript
  // Import any of the ui
  import { Button } from '@onesy/ui-react';

  function A(props) {

    return (
      <Button>
        {props.children}
      </Button>
    );
  }
```

### Dev

Install

```sh
yarn
```

Test

```sh
yarn test
```

### Prod

Build

```sh
yarn build
```
